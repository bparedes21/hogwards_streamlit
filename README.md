# Enterate de que casa sos!


Casa segun suma de los numeros de tu fecha de nacimiento:

- `Ej: 21/11/1990.` 
- `2+1+1+1+1+9+9+0.`
- `Resultado de la suma: 24.`
- `2+4.`
- `Resultado de la suma: 6`
